Kernkonzepte

Workspace = Ein Whiteboard-Raum (z.B. "Mathe Sohn", "Physik Tochter"). Wird vom Lehrer erstellt und Schülern zugewiesen.
Page = Eine einzelne Seite im Workspace. Hat einen Hintergrund-Typ (leer, kariert, liniert, Koordinatensystem). Kann hinzugefügt werden.
Rollen = Lehrer (du) kann alles: erstellen, zuweisen, korrigieren, löschen. Schüler können zeichnen, Seiten hinzufügen, aber keine Workspaces erstellen.
